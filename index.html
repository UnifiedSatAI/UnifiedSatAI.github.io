Unified Deep Learning Benchmark for Satellite Image Restoration and Generation

INTRODUCTION

Modern deep learning methods have significantly advanced the field of satellite image restoration and generation, enabling high-quality cloud removal, super-resolution, and high-definition reconstruction.

However, evaluation practices remain fragmented across datasets and metrics, making comparison difficult.

This project introduces a unified benchmarking framework integrating multiple datasets and standardized preprocessing.

The system evaluates models using PSNR, SSIM, SAM, ERGAS, QNR and FID metrics.

PROJECT

The project develops a unified pipeline for benchmarking satellite image restoration models.

Datasets include AllClear, SEN12MS, WorldStrat, PROBA-V SR and SAT25K.

The pipeline ensures consistent preprocessing and evaluation.
