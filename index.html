INTRODUCTION

43
44 45
46

47 Modern deep learning methods have significantly advanced the field of satellite image restoration and generation, enabling high-quality cloud removal, super-resolution, and high-definition reconstruction from multi-spectral remote sensing data. 48

49 50

51 However, existing evaluation practices remain fragmented, with different datasets, preprocessing pipelines, and metric definitions used across studies. This lack of standardization makes fair comparison between models difficult and limits reproducibility in remote sensing research. 52

53 54

55 The Unified Deep Learning Benchmark for Satellite Image Restoration and Generation is designed to address this problem by providing a consistent and extensible benchmarking framework for evaluating deep learning models across multiple satellite datasets and processing tasks. The system integrates widely used datasets such as AllClear, SEN12MS-CR-TS, WorldStrat, PROBA-V SR, and SAT25K into a single evaluation pipeline with standardized normalization, spectral band alignment, and metric computation. 56

57 58

59 Our framework supports multiple restoration and generation tasks, including cloud removal, super-resolution, and high-definition image synthesis, while ensuring that all experiments follow the same preprocessing rules, evaluation metrics, and reporting structure. In addition to quantitative evaluation using metrics such as PSNR, SSIM, SAM, ERGAS, QNR, and FID, the system provides visualization tools, reproducibility logging, and structured experiment outputs to support reliable scientific analysis. 60

61 62

63 By establishing a unified and reproducible evaluation environment, this project aims to improve transparency, consistency, and comparability in satellite image processing research, and to provide a practical reference framework that can be extended to future datasets, models, and remote sensing applications. 64

PROJECT

75

The project implements a unified benchmarking pipeline for evaluating deep learning models on satellite image restoration and generation tasks.

76 77

The framework supports multiple datasets, including AllClear, SEN12MS-CR-TS, WorldStrat, PROBA-V SR, and SAT25K, and applies consistent preprocessing, normalization, and spectral band alignment to ensure fair comparison across models.

78 79

The system enables evaluation of cloud removal, super-resolution, and high-definition image generation models using standardized metrics such as PSNR, SSIM, SAM, ERGAS, QNR, and FID.
